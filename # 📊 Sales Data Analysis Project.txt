# 📊 Sales Data Analysis Project

## 📌 Objective
The goal of this project is to analyze sales data and find useful insights.

## 🛠 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## 📂 Dataset
The dataset contains sales information like product, price, quantity, and date.

## 📊 Analysis Performed
- Data Cleaning
- Handling missing values
- GroupBy operations
- Data visualization using charts

## 📈 Key Insights
- Highest sales in December
- Top selling product: Laptop
- Region with highest sales: West
